Цепочка сигнала

Источник → 13 полос EQ (±25 дБ, Q на каждой) → Bass Boost → Mono Bass Sum (опц.) → Drive (чистый макап-гейн, без искажений) → Saturation (dry/wet подмес тёплого сатуратора) → Limiter с Ceiling → выход.

Почему теперь не "пердит"

Раньше Drive шёл прямо в агрессивный сатуратор — на высоких значениях это давало жёсткую дисторсию. Теперь Drive — это чистая громкость, а искажение управляется отдельно ползунком Saturation (0-100%, по умолчанию всего 15%). Хочешь погромче и чище — крути Drive и почти не трогай Sat. Хочешь "давящий, чуть подгоревший" саунд для сабвуфера — добавь Sat, но по чуть-чуть.

Сброс

Reset All сверху — сбрасывает весь эквалайзер, Drive/Sat/Ceiling, Bass Boost, Mono Bass. Долгое нажатие (~0.5 сек) на конкретную точку графика без сдвига — сбрасывает только эту полосу в 0.

Mono Bass Sum

Ниже частоты кроссовера левый/правый канал суммируются в один и подаются в оба — бас плотнее и дальше слышен с улицы даже на штатных динамиках.

A/B сравнение

Кнопка "A/B" переключает между обработанным звуком и чистым оригиналом (в обход всей цепочки) — удобно проверить, не переборщил ли ты с настройками.

Пресеты кодом

«Экспорт кода» даёт текстовую строку со всеми настройками — скинь её другу, он вставит в «Импорт по коду» и получит точно такой же звук.

Важное ограничение

Прямые ссылки на треки в Spotify и SoundCloud здесь не подключаются — оба сервиса не отдают сырой аудио-поток в браузер (DRM у Spotify, ограничения потокового API у SoundCloud). Работает с загруженными файлами и прямыми ссылками на аудио.

В веб-версии обрабатывается один локальный источник звука за раз. Мультивкладочный захват — функция расширения Chrome (Pro).

Roadmap: что портируем из EquaTelegram дальше

Часть функций твоего бота можно перенести прямо сюда (клиентский код), часть требует бэкенда — как у бота (сервер + БД + ffmpeg). Разложил по приоритету:

Waveform-превью трекаnext
рисуем форму волны файла на канвасе под плеером — чисто клиентский код, Web Audio decodeAudioData.
Многополосный компрессор (низ/середина/верх раздельно)next
сейчас один общий лимитер на выходе; разносим на 3 полосы своими компрессорами — даст более "клубный", контролируемый бас без просадки середины.
Fade-in/out + обрезка тишиныnext
клиентский код поверх Web Audio, не требует сервера.
Тёмная/красная/светлая темаnext
3 темы как в боте, переключатель сверху.
Стерео-расширение, De-essernext
StereoPannerNode/Mid-Side трюк для расширения + узкополосный компрессор на 4-8кГц под de-esser.
Библиотека, плейлисты, история прослушиванийнужен бэкенд
как в bassbot.db — без сервера с БД это не переживёт закрытие вкладки.
Экспорт готового файла (рендер через ffmpeg)нужен бэкенд
браузер не может рендерить mp3 с эффектами в файл так же, как ffmpeg на сервере — тут нужен твой бэкенд как в EquaTelegram.
Групповая обработка нескольких треков разомнужен бэкенд
то же самое — требует очереди задач и хранилища файлов.
